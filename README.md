# Kick of the Fruits v0.1

明るいメルヘン世界で、魔法拳法少女がフルーツモンスターと戦うHTML5ゲームです。

敵：
- いちご：下段
- メロン：下段〜中段をバウンド
- ぶどう：中段
- レモン：中段
- パイナップル：上段
- スイカ：最上段

MAGIC 100%でレインボーマジックを発動します。
必殺技中の撃破ではMAGICゲージは増えません。
