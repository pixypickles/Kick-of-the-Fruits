# Kick of the Fruits v1.0

ゼロから組み直した安定版です。

## 主な仕様

- EASY / NORMAL / HARD
- 下段・中段・上段・ジャンプ攻撃
- いちご、メロン、ぶどう、レモン、パイナップル、スイカ
- メロンは下段〜中段をバウンド
- MAGIC 100%でレインボーマジック
- 必殺技中に倒した敵からMAGICゲージは増えません
- 開始時に3秒のREADY無敵時間
- 主人公は以前より小さめに表示

## GitHub Pages

ZIPを展開し、中のファイルをすべてリポジトリ直下へアップロードしてください。
旧バージョンのファイルは削除または上書きしてください。


## v1.1 の変更

- ミドルキックとハイキックで同じ開始モーションを使用
- 質感と解像感を統一
- ヒット時のミドルキック画像・ハイキック画像はそれぞれ従来どおり


## v1.2 の変更

- 新しいミドルキック画像を採用
- 主人公表示サイズを元の400pxへ戻し、高さ判定と見た目を合わせやすく調整
- ジャンプキックは、足の現在位置と敵の高さが合っていれば全レーンへ命中
- ジャンプキックの最大巻き込み数を5体へ増加
- 必殺技を「RAINBOW FRUIT STORM」に刷新
- 画面暗転、回転魔法陣、虹色衝撃波、流星、クリスタル、中央フラッシュを追加
- 必殺技中の撃破でMAGICゲージが増えない仕様は維持


## v1.3
- 細い折りたたみスマホでも3難易度を同時表示
- 下段蹴りに途中モーションを追加
- 全モーションを共通サイズ・共通足元位置に正規化
- モーション切替時の顔や体の大きさの差を軽減


## v1.4 — 新しい統一キャラクター素材

- 新ニュートラル画像を基準として採用
- ミドルキック：構え／ヒットを新画像へ変更
- ハイキック：ミドルと共通の構え／専用ヒット画像
- ジャンプ：空中構え／ジャンプキック専用ヒット画像
- 下段キック：専用構え／専用ヒット画像
- 必殺技：チャージ画像 → 発動画像へ切り替え
- ダメージ専用画像と短い硬直を追加
- 全素材を顔の大きさと足元基準で再配置
- ステージクリアや勝利ポーズはありません
- ゲーム形式はスコアを競うエンドレスサバイバルです


## v1.5 — モーションサイズの再統一

- 中蹴りでキャラクターが急に大きくなる問題を修正
- 全画像を「全身の高さ」ではなく「頭・顔の横幅」を基準に再調整
- 横に長いキック画像を無理に縮小しない方式へ変更
- すべて960×960の共通透明キャンバスへ配置
- ゲーム内では全モーションを同じ描画サイズで表示
- 各フレームの足元基準を統一


## v1.6 — 個別倍率固定版

- 自動サイズ補正を廃止
- モーションごとにゲーム専用の固定倍率・固定位置を設定
- 新しく拡大した中蹴り、ジャンプ、ジャンプキック画像を採用
- 中蹴りヒットを基準より約5%大きく調整
- ジャンプを基準より約3.5%大きく調整
- ジャンプキックを基準より約6.5%大きく調整
- 全画像を960×960の共通キャンバスへ配置
- ゲーム内の描画矩形は全モーション共通
- 今後は tuning 表の数値だけで個別に微調整可能
